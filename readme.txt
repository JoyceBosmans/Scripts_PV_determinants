This repository contains scripts used for the ERL paper "Determinants of the distribution of utility-scale photovoltaic power facilities across the globe", by Joyce Bosmans et al, accepted in October 2022. 

The directories SolarWindPotentials_Global and PV_determinants_* contain scripts used to gather and preprocess data. 
Modelling was done with scripts in PV_code_INLA. 
Processing of results was done with the scripts in directories PV_Konrad_*
